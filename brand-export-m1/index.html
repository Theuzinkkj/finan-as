7 · Stripe Dashboard

Onde colocar o ícone na Stripe:
Dashboard → Settings → Branding → Icon (logo da empresa nas faturas/recibos). Use stripe-icon-256.png ou stripe-icon-512.png (Stripe redimensiona automaticamente).

Cores de marca pra Stripe: Primary #7C5CFF · Secondary #0B0B16.

Arquivos no pack

  • 📐 favicon.svg — vetor, recomendado (suporta retina ilimitado)
  • 🖼 favicon-{16,32,48,64}.png — fallback PNG
  • 🍎 apple-touch-icon.png — 180×180 com fundo escuro
  • 📱 icon-{192,512}.png — PWA / Android
  • 💳 stripe-icon-{256,512}.png — pra upload no Stripe Dashboard
  • 🎨 logo-mark.svg, logo-mark-white.svg, logo-mark-black.svg — símbolo só
  • 🎨 logo-mark-{128,256,512}.png — símbolo em raster
  • 🏷 logo-full.svg, logo-full-dark-text.svg — logo + texto
  • 🏷 logo-full-white-{1x,2x,3x}.png, logo-full-dark-{1x,2x,3x}.png
