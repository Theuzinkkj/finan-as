Perguntas frequentes

Esqueci minha senha. Como recupero o acesso?
Na tela de login, clique em "Esqueceu a senha?". Você receberá um e-mail com um link para criar uma nova senha. Verifique também a pasta de spam. O link é válido por 30 minutos.
Como assino o plano Pro?
Acesse Configurações → Plano no app, ou clique em "Ver planos" no menu. Você pode assinar via cartão de crédito ou PIX. Assim que o pagamento for confirmado, os recursos Pro são liberados imediatamente.
Posso cancelar o plano Pro a qualquer momento?
Sim. Cancele em Configurações → Plano → Cancelar assinatura. O acesso Pro permanece ativo até o fim do período já pago. Não há multa por cancelamento.
Como exporto meus dados financeiros?
Na tela principal, toque no ícone de menu (⋯) e selecione "Exportar". Você pode exportar em CSV (compatível com Excel e Google Sheets) ou PDF. Usuários do plano Gratuito exportam o mês atual; o plano Pro permite exportar qualquer intervalo de datas.
O assistente de IA tem acesso a todas as minhas transações?
O assistente recebe apenas um resumo agregado das suas finanças do mês (totais por categoria, saldo geral), nunca os nomes individuais das transações nem dados pessoais. Leia nossa Política de Privacidade para mais detalhes.
Como atualizo o Atlas para a versão mais recente?
O Atlas é um aplicativo web — as atualizações chegam automaticamente quando você abre o app. Se algo parecer desatualizado, force a recarga da página (Ctrl + Shift + R no desktop, ou "Limpar dados do site" no celular).
Como excluo minha conta e todos os meus dados?
Acesse Configurações → Conta → Excluir conta. Todos os seus dados são removidos permanentemente dos nossos servidores em até 30 dias. Exporte seus dados antes, pois a exclusão é irreversível.
O que é o modo demo?
O modo demo permite explorar o Atlas sem criar uma conta. Os dados ficam salvos apenas no seu dispositivo e são apagados ao sair. É ótimo para conhecer o app antes de se cadastrar.
Posso receber lembretes para lançar transações?
Sim. Em Configurações → Notificações, ative os lembretes diários ou semanais. Você define o horário preferido. Os lembretes precisam de permissão de notificação no navegador.
Encontrei um bug. Como reporto?
Envie um e-mail para suporte@atlasfinance.app descrevendo o que aconteceu, em qual dispositivo/navegador e, se possível, um print da tela. Quanto mais detalhes, mais rápido conseguimos resolver.
